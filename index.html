    Contraceptive Method
    Use Analysis Project

    This project analyzes the contraceptive method choice dataset which is a subset of the 1987 National Indonesia Contraceptive Prevalence Survey (archive.ics.uci.edu). We will use various predictive models including Logistic Regression, Decision Trees, Support Vector Machines, k-Nearest Neighbors, Cluster Analysis, and Random Forests, we to predict if a woman will use no contraceptives, short-term contraceptives, or long-term contraceptives based off a variety of predictor variables.
